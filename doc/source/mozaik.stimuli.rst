stimuli Package
===============

This sub-package contains modules that are related to definition and manipulation of stimuli.

Modules
^^^^^^^    
    * :py:mod:`mozaik.stimuli.stimulus` - contains definition of the stimulus API 
    * :py:mod:`mozaik.stimuli.visual_stimulus` - contains definitions of the visual stimulus API, and implements some of the functions specified in the stimulus interface
    * :py:mod:`mozaik.stimuli.topographica_based` - contains implementation of number of visual stimuli, using the `Topographica <http://topographica.org/>`_ pattern generation package.

See the :py:mod:`mozaik.stimuli.stimulus` for more general information on stimulus handling in *mozaik*.
